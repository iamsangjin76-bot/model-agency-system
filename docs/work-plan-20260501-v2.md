## 새로 발견된 이슈

### 🔴 이슈 1 — PPT 내보내기 완전 불동작 (python-pptx 미설치)

**원인:** `backend/requirements.txt`에 `python-pptx` 없음  
**영향:** PPT 내보내기 클릭 시 HTTP 503 "python-pptx가 설치되지 않았습니다" 응답  
**발견 근거:** `export.py:64` lazy import + ImportError catch → 503 반환 코드 확인  

**수정 파일:** `backend/requirements.txt`  
**수정 내용:** `python-pptx>=0.6.21` 한 줄 추가

---

### 🟡 이슈 2 — 백업 상태 API 버그 (BACKUP_SECRET NameError)

**영향:** `/api/backup/status` 엔드포인트 호출 시 `NameError` 500 오류  
**주 기능(다운로드)은 무관 — status 엔드포인트만 영향**

**수정 파일:** `backend/app/routers/backup.py`  
**수정 내용:** `bool(BACKUP_SECRET)` → `True` (JWT 인증 자체가 보안 수단)
