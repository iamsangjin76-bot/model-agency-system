### 4. Lore 훅의 실제 검증 범위 (중요!)

explorer 조사 결과:
- **검증함**: type prefix 형식 + subject line 길이(≤72자)
- **검증 안 함**: Autopus sign-off, Constraint 존재 여부, trailer 값 열거
- `auto lore validate` 명령은 문서엔 있지만 **구현 스크립트는 repo 에 없음**

이는 현재 훅이 상당히 허술하다는 뜻. 장래 훅 강화 시 우리가 쓴 값들이 갑자기 거부될 수 있음. **처음부터 정답을 쓰는 게 안전.**

### 5. PowerShell 에서 커밋 메시지 파일 관리

- 커밋 연속 진행 시 임시 파일 삭제하지 말고 **내용만 교체**해서 재사용
