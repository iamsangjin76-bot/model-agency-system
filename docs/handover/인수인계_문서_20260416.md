## ⚠️ 주의사항 / 알려진 이슈

1. **GitHub 히스토리에 과거 민감 파일이 남아있음**
   - `.env` (플레이스홀더 시크릿), `venv/`, `.db` 가 initial commit 에 포함
   - 현재 플레이스홀더였고 Private repo 라 실질 위험은 낮음
   - 완벽한 정리 원하면 `BFG Repo-Cleaner` 로 히스토리 재작성 가능 (지금은 비권장)

2. **대시보드 "데이터 없음" 표시는 정상 동작**
   - DB 에 실제 데이터가 없어서 차트가 빈 상태
   - 버그 아님, 정확한 UX 처리
   - 해결 = 시드 데이터 투입 (우선순위 7)

3. **"최근 활동" 패널은 하드코딩 더미 데이터**
   - "김모델 / 홍길동 / 박지민 / 이수진" 은 화면 확인용 예시
   - 실제 DB 로 연결 필요 (우선순위 8)

4. **requirements.txt 에 httpx 누락**
   - 수동으로 `pip install httpx` 했음
   - 다음 세션에 `requirements.txt` 업데이트 + 커밋 필요

5. **절대 Claude Code 가 `git push`/`git reset --hard`/`git filter-branch` 같은 **되돌리기 어려운 명령**을 먼저 실행하도록 하지 말 것.** 반드시 사용자 허가 후.
