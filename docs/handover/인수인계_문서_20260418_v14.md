## 🎓 이번 세션에서 얻은 교훈

### 1. reviewer가 xl col-span 누락을 잡아냄 🆕
- 부모를 `xl:grid-cols-4`로 바꾸면서 자식의 `col-span` 합이 3에서 멈춘 채로 놓침
- reviewer 리뷰가 없었으면 xl에서 4번째 컬럼이 빈 채로 배포될 뻔함
- **교훈**: grid-cols 변경 시 반드시 자식 col-span 합 = 부모 cols 검증

### 2. 다크모드 텍스트 색상은 배경보다 중요 🆕
- `dark:bg-gray-800`만 추가하고 텍스트 `dark:text-*`를 빠뜨리면 글자가 안 보임
- 배경과 텍스트는 반드시 쌍으로 작업해야 함
- **교훈**: dark: 작업 시 bg + text + border를 세트로 처리

### 3. deep-worker의 대규모 일괄 처리 효과 🆕
- 36개 파일 802줄을 약 18분 만에 처리
- 단, 자동 커밋+push까지 해버려서 중간 검증 기회가 없었음
- **교훈**: deep-worker 사용 시 자동 push 방지 지시를 Contract에 명시

### 4. Tailwind lg:grid-cols-2 중복 선언은 무해하지만 노이즈 🆕
- `grid-cols-2 lg:grid-cols-2`는 기능상 동일하지만 코드 가독성 저하
- reviewer가 WARN으로 잡아서 정리 → 깔끔한 코드 유지
