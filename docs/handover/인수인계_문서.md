## 🔧 새 창에서 바로 해야 할 것

### Step 1 — 관리자 PowerShell 2개 열기
- 창 1: 프론트엔드 (npm run dev)
- 창 2: 백엔드 (uvicorn)

### Step 2 — 창 1에서 프론트엔드 실행
```
cd "F:\Project M\model-agency-system\frontend"
npm run dev
```

### Step 3 — 창 2에서 백엔드 실행
```
cd "F:\Project M\model-agency-system"
python -m uvicorn backend.app.main:app --reload
```

### Step 4 — 브라우저에서 확인
- http://localhost:5173 접속
- 로그인 테스트 (admin 계정)
- 대시보드 차트 3개 확인
